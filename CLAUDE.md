# Waqas Ali - Portfolio Project

This project is a professional portfolio landing page for Waqas Ali, built with Next.js and Tailwind CSS.

## Project Structure

This file provides a quick reference to the key files and their purpose.

### Core Application Logic
- `src/app/layout.tsx`: The main layout file, sets up the HTML structure and dark theme.
- `src/app/page.tsx`: The entry point of the application, assembling all the components.
- `src/app/globals.css`: Global CSS styles and Tailwind directives.

### Configuration
- `package.json`: Lists project dependencies and scripts.
- `tailwind.config.ts`: Configuration for the Tailwind CSS framework.
- `next.config.mjs`: Configuration for the Next.js framework.
- `tsconfig.json`: TypeScript configuration, including path aliases.

### Content & Components (Documentation)
*All application components are located in the `/docs` directory.*

- `docs/Header.tsx`: The site's navigation header.
- `docs/Hero.tsx`: The main hero section with the introductory message.
- `docs/About.tsx`: The "About Me" section.
- `docs/Projects.tsx`: The portfolio projects showcase.
- `docs/Skills.tsx`: The skills and expertise section.
- `docs/Experience.tsx`: The professional experience timeline.
- `docs/Contact.tsx`: The contact information and form.
- `docs/Footer.tsx`: The site's footer.
